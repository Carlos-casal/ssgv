# Changelog

## [1.6.0] - 2024-05-24
### Added
- **Multi-Batch System:** Support for multiple lots (batch number, expiry, supplier, purchase price, margin) per consumable material.
- **FIFO Logic:** Automated inventory consumption prioritizing oldest batches by expiration date.
- **Persistent Notifications:** Database-backed system for stock and expiration alerts with "Mark as Read" and "Delete" actions.
- **Dark Mode Overhaul:** Global CSS variable system for robust theme support and high-contrast UI components.
- **Activity Log & Alerts Sync:** Optimization of system status updates on the main dashboard.

## [1.5.0] - 2024-05-15
### Added
- Excel import/export system for materials.
- Dynamic form validation for volunteers and services.

## [1.0.0] - 2024-03-20
### Added
- Initial release with core volunteer and service management.
