# SSGV Project

## Overview
This README contains detailed documentation about the SSGV project, which aims to provide various features for our users.

## New Features
### Warehouse Traceability System
We've implemented a comprehensive warehouse traceability system, allowing users to track items throughout the entire supply chain. This system provides:
- Real-time inventory tracking
- Detailed audit logs for each item
- Alerts for low stock levels

### Service Form Redesign
The service form has been redesigned for a better user experience. Key improvements include:

## Getting Started
- Clone the repository:
  ```bash
  git clone https://github.com/Carlos-casal/ssgv.git
  ```
- Install the necessary dependencies:
  ```bash
  cd ssgv
  npm install
  ```
- Start the development server:
  ```bash
  npm start
  ```

For more detailed instructions, please refer to the documentation in each module.
